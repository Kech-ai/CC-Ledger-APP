MERN Application Ready

This application is now a full MERN stack application and must be run from the backend server.

1. Open a terminal in the project's root directory.

2. Navigate to the server: cd server

3. Install dependencies: npm install

4. Start the server: npm start

Then, open your browser and navigate to http://localhost:5000.

The old frontend-only files in the root directory (like this one's predecessor) are now obsolete and should be deleted.
